# Мобильный сканер кодов

Одностраничное Nuxt 4 приложение, которое получает изображение с основной камеры,
рисует его в `canvas` и непрерывно распознаёт товарные и двумерные коды.

Распознанный результат передаётся родительскому компоненту только после нажатия
кнопки «Сканировать». Одно нажатие выдаёт ровно один результат.

## Запуск

```bash
npm install
npm run dev
```

Локальный адрес: `http://localhost:3000`.

На телефоне доступ к камере требует HTTPS. Исключение браузеры делают только для
`localhost`, поэтому адрес компьютера в локальной сети без сертификата не подойдёт.

## Статическая сборка

Приложение работает как клиентская SPA без Node-сервера. Готовые к публикации
файлы создаются в `.output/public`:

```bash
npm run generate
```

Push в ветку `main` запускает `.github/workflows/deploy-pages.yml`, который
публикует этот каталог в Cloudflare Pages. Репозиторию нужны GitHub Actions
secrets `CLOUDFLARE_API_TOKEN` и `CLOUDFLARE_ACCOUNT_ID`; Pages-проект должен
называться `flat-wave-d403`, а его production branch — `main`.

## Проверки

```bash
npm test
npm run typecheck
npm run build
```

Тесты не зависят от интерфейса. Они проверяют автоматы камеры и затвора,
геометрию видеокадра, ограничение частоты анализа, защиту от параллельного
декодирования и настоящее чтение QR-кода из RGBA-пикселей через ZXing.

## Устройство сканера

- `app/scanner/domain` — типы и конечные автоматы без побочных эффектов.
- `app/scanner/core` — обработка геометрии кадра и правило выдачи результата.
- `app/scanner/infrastructure` — связь основного потока с декодером.
- `app/scanner/workers` — распознавание в отдельном потоке через ZXing/WASM.
- `app/composables` — жизненный цикл камеры и отрисовка видео в `canvas`.
- `app/components/CameraScanner.vue` — переиспользуемый интерфейс сканера.

Компонент отправляет событие `scan`:

```ts
interface ScanResult {
  value: string
  format: string
  scannedAt: string
}
```

По умолчанию используется метаформат `AllReadable`. Ограничить или расширить
список можно через `DEFAULT_BARCODE_FORMATS` в
`app/scanner/domain/scanner.types.ts`; остальная архитектура от форматов не зависит.
